 ## TJA Editor
 TJA Editorは、太鼓さん次郎用の TJA 譜面を編集するための
Windows向けエディタです。

譜面制作・調整・プレビューを一つの画面で行えることを目的に、
実制作向けの機能と安定性を重視して開発しています。

## 特徴
- TJAファイルの編集に特化したシンプルなUI
- ダークモード / ライトモード対応
- 太鼓さん次郎と連携したプレビュー再生（F5）
- 編集中ファイルの自動バックアップ機能
- 最近使ったファイル・最終フォルダの記憶
- ステータスバーによる編集状況の表示

## 動作環境
- Windows 10 / Windows 11
- 太鼓さん次郎（別途必要）

## 使い方
1. Releases から zip ファイルをダウンロード
2. 解凍して TJAEditor.exe を実行
3. TJAファイルを開く、または新規作成
4. F5キーでプレビュー再生

※ 初回プレビュー時は Taikojiro.exe の場所を指定してください。
